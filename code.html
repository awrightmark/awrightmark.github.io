Minimum Convex Hull Mass and Center of Mass Calculation

Link to GitHub repository.

This Python script takes body segments that have already been separated onto individual surface mesh files, wraps each in its own convex hull, and computes the mass and center of mass for each segment as well as for the whole body.

This script was used in the following publicatoin:

  • Wright MA, Cavanaugh TJ, Pierce SE (2024). Volumetric versus Element-scaling Mass Estimation and Its Application to Permo-Triassic Tetrapods. Integrative Organismal Biology. Link

Correcting Inconsistent Euler XYZ Rotation Orders

Link to GitHub repository.

This R script corrects inconsistent Euler XYZ rotation orders across a data file by converting Euler coordinates into quaternion notation, which is independent of rotation order, and then converting quaternion notation back into Euler XYZ coordinates. The design provides seamless integration with existing Maya data processing workflow.

This script was used for Chapter 2 of my dissertation, which will ideally be submitted and approved by May 2025.

Comparing Morphogenetic Shape Data and Trajectories

Link to GitHub repository.

These R functions assist in the calulation of morphogenetic trajectories for developmental series on morphospaces and provide pairwise Euclidean distance calculations for a priori defined groups or individuals on the morphospace.

This script was used for Chapter 3 of my dissertation, which will ideally be submitted and approved by May 2025.
